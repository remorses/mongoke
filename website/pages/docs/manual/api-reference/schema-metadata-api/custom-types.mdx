
Schema/Metadata API Reference: Custom Types
===========================================

**Custom Types** are user-defined GraphQL types which help to define
`Actions <api_actions>`{.interpreted-text role="ref"}.

set\_custom\_types
------------------

`set_custom_types` is used to set user-defined GraphQL types. This API
will replace the given types with existing ones.

```http
POST /v1/query HTTP/1.1
Content-Type: application/json
X-Hasura-Role: admin

{
  "type": "set_custom_types",
  "args": {
    "scalars": [],
    "enums": [],
    "input_objects": [
      {
        "name": "User",
        "fields": [
          {
            "name": "username",
            "type": "String!"
          },
          {
            "name": "password",
            "type": "String!"
          }
        ]
      }
    ],
    "objects": [
      {
        "name": "UserId",
        "fields": [
          {
            "name": "id",
            "type": "Int!"
          }
        ],
        "relationships": [
          {
            "name": "posts",
            "type": "array",
            "remote_table": "post",
            "field_mapping": {
              "id": "user_id"
            }
          }
        ]
      }
    ]
  }
}
```

### Args syntax

  Key              Required   Schema                                         Description
  ---------------- ---------- ---------------------------------------------- -------------------------------
  input\_objects   false      Array of [InputObjectType](#inputobjecttype)   Set of GraphQL `Input Object`
  objects          false      Array of [ObjectType](#objecttype)             Set of GraphQL `Object`
  scalars          false      Array of [ScalarType](#scalartype)             Set of GraphQL `Scalar`
  enums            false      Array of [EnumType](#enumtype)                 Set of GraphQL `Enum`

#### InputObjectType

A simple JSON object to define [GraphQL Input
Object](https://spec.graphql.org/June2018/#sec-Input-Objects)

  Key           Required   Schema                                           Description
  ------------- ---------- ------------------------------------------------ --------------------------------------
  name          true       `GraphQLName`{.interpreted-text role="ref"}      Name of the Input object type
  description   false      String                                           Description of the Input object type
  fields        true       Array of [InputObjectField](#inputobjectfield)   Fields of the Input object type

##### InputObjectField

  Key           Required   Schema                                                      Description
  ------------- ---------- ----------------------------------------------------------- ---------------------------------------
  name          true       `GraphQLName`{.interpreted-text role="ref"}                 Name of the Input object field
  description   false      String                                                      Description of the Input object field
  type          true       `GraphQLType <GraphQLType>`{.interpreted-text role="ref"}   GraphQL ype of the input object field

#### ObjectType

A simple JSON object to define [GraphQL
Object](https://spec.graphql.org/June2018/#sec-Objects)

  Key             Required   Schema                                               Description
  --------------- ---------- ---------------------------------------------------- --------------------------------------------
  name            true       `GraphQLName`{.interpreted-text role="ref"}          Name of the Object type
  description     false      String                                               Description of the Object type
  fields          true       Array of [ObjectField](#objectfield)                 Fields of the Object type
  relationships   false      Array of [ObjectRelationship](#objectrelationship)   Relationships of the Object type to tables

##### ObjectField

  Key           Required   Schema                                                      Description
  ------------- ---------- ----------------------------------------------------------- ----------------------------------------
  name          true       `GraphQLName`{.interpreted-text role="ref"}                 Name of the Input object field
  description   false      String                                                      Description of the Input object field
  type          true       `GraphQLType <GraphQLType>`{.interpreted-text role="ref"}   GraphQL type of the input object field

##### ObjectRelationship

  Key              Required   Schema                                                                                                 Description
  ---------------- ---------- ------------------------------------------------------------------------------------------------------ -------------------------------------------------------------------------
  name             true       `RelationshipName`{.interpreted-text role="ref"}                                                       Name of the relationship, shouldn\'t conflict with existing field names
  type             true       \[ `object` \| `array` \]                                                                              Type of the relationship
  remote\_table    true       `TableName`{.interpreted-text role="ref"}                                                              The table to which relationship is defined
  field\_mapping   true       Object ([ObjectField](#objectfield) name : Remote table\'s `PGColumn`{.interpreted-text role="ref"})   Mapping of fields of object type to columns of remote table

#### ScalarType

A simple JSON object to define [GraphQL
Scalar](https://spec.graphql.org/June2018/#sec-Scalars)

  Key           Required   Schema                                        Description
  ------------- ---------- --------------------------------------------- --------------------------------
  name          true       `GraphQLName`{.interpreted-text role="ref"}   Name of the Scalar type
  description   false      String                                        Description of the Scalar type

#### EnumType

A simple JSON object to define [GraphQL
Enum](https://spec.graphql.org/June2018/#sec-Enums)

  Key           Required   Schema                                        Description
  ------------- ---------- --------------------------------------------- ------------------------------
  name          true       `GraphQLName`{.interpreted-text role="ref"}   Name of the Enum type
  description   false      String                                        Description of the Enum type
  values        true       Array of [EnumValue](#enumvalue)              Values of the Enum type

##### EnumValue

  Key              Required   Schema                                        Description
  ---------------- ---------- --------------------------------------------- ----------------------------------------------------------
  value            true       `GraphQLName`{.interpreted-text role="ref"}   Value of the Enum type
  description      false      String                                        Description of the value
  is\_deprecated   false      Boolean                                       If set to `true`, the enum value is marked as deprecated
